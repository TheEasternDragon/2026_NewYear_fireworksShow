<h22.怎么运行？></h2>
<p>运行本项目需要配置JavaScript的环境进行用Nodejs即可</p>
<p>大家可以在此链接下载：<a href="https://nodejs.org/">nodeJS下载</a></p>
<div align="center"><img src="img/nodeJS.png" alt="图片"></div>
<p>下载完成后，用VScode打开项目，在终端运行<code>npm install express</code>来安装express</p>
<p>现在可以开启服务器了，终端输入<code>node server.js</code>(路径不唯一，确保输入正确的server.js文件路径)</p>
<p>在终端，你会看到如下输出：</p>
<p><code>服务器已经启动...</code></p>
<p><code>本机访问：http://localhost:3000</code></p>
<p><code>局域网访问：http://x.x.x.x:3000</code></p>
<p>自己访问的话，就可以访问http://localhost:3000</p>
<p>发给家人就可以用下面那个了（只能在局域网内哈），还是挺有意思吧Ψ(￣∀￣)Ψ</p>
